Git


Git - розподіленя система керування версіями файлів. Git є однією з найефективніших, надійних і високопродуктивних систем керування версіями, що надає гнучкі засоби недінійною розробки, що базуються на відгалуженні і злитті гілок

GitHub

Пост на Сайті|Бер 20, 2011|4 коментарі

GitHub - один з найбільших веб-сервісів для спільної розробки програмного забезпечення. Існують безкоштовні та платні тарифні плани користування сайтом. Бaзується на системі керування версіями Git і розроблений на Ruby on Rails і Erlang компанією GitHub, Inc(ранішеLogical Awesome).

24 лютого 2009 року в бесіді, яка відбувалась в штаб-квартирі !, команда GitHub оголосила, що протягом першого року перебуванняв режимі онлайн, GitHub накопичив понад 46 000 публічних репозитаріїв, з яких 17 000 були створені лише за попередній місяць. У той час близько 6 200 репозитаріїв принаймі один раз робили форк, 1 4600 робили мердж (злиття двох гілое в одну). 23 грудня 2013 року компанія GitHub оголосила що кількість репозиторіїв становить 10 мільйонів.
